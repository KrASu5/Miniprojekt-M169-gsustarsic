Arbeitsauftrag: Projektrollen und deren Aufgaben, Kompetenzen und Verantwortlichkeiten

1) Überlegen Sie sich drei verschiedene Rollen in einem Projekt und beschreiben Sie diese.

2) Ergänzen Sie zu jeder gefundenen Rolle eine Tabelle mit den Aufgaben, Kompetenzen und Verantwortungen.
In einem Projekt, dass wir im BLJ durchgeführt haben, bestand aus drei Rollen:
  • Zuständiger Servermanager für Firma A
  • Zuständiger Servermanager für Firma B
  • Zuständiger Servermanager für Firma C
Eine grosse IT-Firma (Firma A) hat zwei kleinere Hardware-Distributoren (Firma B und C) im Ausland gekauft. Nun gilt es die Firmen IT-technisch zusammenzuführen. Jede der 3 Firmen bietet eigenen Serverdienste an, die teilweise von den anderen Firmen benutzt werden sollen.

Ich beschreibe nun die Tätigkeiten dieser drei Rollen:

Zuständiger Servermanager für Firma A

Er betreibt den Domänencontroller der Firma A, der auch Server für Druck- und Dateifreigaben ist.

Zuständiger Servermanager für Firma B

Er betreibt einen Webserver auf den alle Mitarbeiter der Hauptfirma A zugreifen können. Auch die Domäne der Firma C soll als Subdomain an die Domain der Firma A angeschlossen werden.

Zuständiger Servermanager für Firma C

Er betreibt zwei Webserver auf den alle Mitarbeiter der Hauptfirma A ebenfalls zugreifen können.

Zuständiger Servermanager für Firma A:
Aufgabe -einen Domänencontroller aufsetzen
-Server für Druck- und Dateifreigaben aufsetzen.
Kompetenz -kann Domänencontroller aufsetzen
-kann Server für Druck- und Dateifreigaben aufsetzen
Verantwortung -ist Verantwortlich für sämtliche Prozesse der Firma A


Zuständiger Servermanager für Firma B:
Aufgabe -hängt die Firma B mittels Subdomain an die Firma
-erstellt einen Webserver, auf den alle Mitarbeiter der Firma A angeschlossen werden
-Die Domäne der Firma C soll ebenfalls mittels Subdomain angeschlossen werden
Kompetenz -Kann Server an eine Subdomain hängen
-Kann einen Webserver aufsetzen
Verantwortung -ist Verantwortlich für sämtliche Prozesse der Firma B


Zuständiger Servermanager für Firma C:
Aufgabe -betreibt zwei Webserver auf den alle Mitarbeiter der Hauptfirma A ebenfalls zugreifen können
Kompetenz -Kann Webserver aufsetzen
Verantwortung -ist Verantwortlich für sämtliche Prozesse der Firma C
